Assignment 4, Game created by Chloe Han

Hey! This is the website for my assignment 4, the Robot Factory.
You are able to interact with the direction buttons to direct the robots anywhere you want.
Aside from just setting directions, I've added some extra features to add more fun.
First, there are limits to the number of times you can change the direction of each arrow. It is set by 10 as a default, but few arrows were randomly set to be fixed, where you won't be able to change their directions. The number of times left for each button will be shown on the top left of each arrow. The arrow will be drawn with a orange edge if it is no longer changeable.
Also, there are 2 pink doors on the right, where you will want your robots to reach at the end of their journey.
You will earn a score every time a robot passes the door.
Although it is a simple game, I'm sure you're gonna enjoy it!
